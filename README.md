## Text Summarizer using LangChain and Gemini

# Project Overview
This project is a Generative AI application that summarizes long text into concise and meaningful summaries using Google's Gemini model and LangChain. The user provides a paragraph, article, notes, or any textual content, and the system generates a short summary highlighting the key information.

# Objectives
1. Generate concise summaries from lengthy text.
2. Learn the basics of Generative AI applications.
3. Understand Prompt Engineering using LangChain.
4. Integrate Google's Gemini API with LangChain.
5. Build an end-to-end AI-powered text processing application

# Technologies Used
1. Python
2. LangChain
3. Google Gemini API
4. Jupyter Notebook
5. VS Code

# Features
1. Accepts user-provided text as input.
2. Generates accurate and concise summaries.
3. Uses Prompt Templates for structured prompting.
4. Supports long paragraphs and articles.
5. Simple and beginner-friendly implementation.

# Project Workflow
1. User enters text.
2. LangChain Prompt Template formats the input.
3. Gemini model processes the text.
4. AI generates a summary.
5. Summary is displayed to the user.

# Workflow Diagram
User Input → Prompt Template → Gemini Model → Generated Summary

# API Configuration
Generate a Gemini API key from Google AI Studio and add it to the project.
